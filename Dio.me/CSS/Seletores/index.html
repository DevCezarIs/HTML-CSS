Css tem certo tipos de seletores

  • Seletor por Tipo: Busca elementos por uma tag HTML especifica
  • Seletor por Id(#):Busca elementos atraves di atri uto 'id'
  • Seletor por Classe(.):Busca elementos pelo atributo 'class
  • Seletor Universal(*)Seleciona todos os elementos HTML
  • Seletor por atributo([atrib])Seleciona elementos com atributos especificos
